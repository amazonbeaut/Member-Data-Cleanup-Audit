# Member-Data-Cleanup-Audit
Take a messy list of member contacts and "clean" it so it can be used for a professional mailing list or loan report.

Member Data Cleanup & Audit
Technical Stack: SQL (Basic Queries, String Functions, Aggregations)

The Business Problem
Working in financial services, I’ve seen how inconsistent data entry can lead to errors in loan processing or member outreach. I created this script to automate the "audit" process of a raw member database.
